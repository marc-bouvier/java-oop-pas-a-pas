# Java OOP pas à pas

Dans [ce dépôt](https://github.com/marc-bouvier/java-oop-pas-a-pas), vous trouverez 
des exercices ainsi que leurs corrigés 
pour découvrir une partie de la syntaxe du langage Java.

Vous pouvez aussi ouvrir les instructions dans votre navigateur : [marc-bouvier.github.io/java-oop-pas-a-pas](https://marc-bouvier.github.io/java-oop-pas-a-pas/)

Certains concepts de la programmation orientée objet
seront appliqués.

> "Programmation Orientée Objet" sera parfois abbrégée 
> en "POO" (ou "OOP" en anglais)

Cet ensemble d'exercices accompagne le site du cours que
vous trouverez à l'adresse suivante : https://bc.baldir.fr

> ‼️ Ces exercices nécessitent d'utiliser **IntelliJ Idea Ultimate**

## Organisation des exercices

Chaque étape est située dans son propre dossier.

Par exemple, l'étape 01 est dans le dossier [etape-01](etape-01).

### Structure d'une étape

Chaque étape est structurée de la façon suivante :

```text
.
├── README.md           - Énoncé de l'exercice
│
│
├── exo                 - Dossier dans lequel résoudre l'exercice
│   ├── pom.xml
│   └── src
│       └── main
│           └── java    - Là où on écrit le code java (fichiers .java)
│
│
└── correction          - Dossier de la correction
    ├── pom.xml
    └── src
        └── main
            └── java    
```
